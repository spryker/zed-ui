# ZedUi Module
[![Build Status](https://travis-ci.org/spryker/zed-ui.svg)](https://travis-ci.org/spryker/zed-ui)

This module provides base UI components for Zed application.

## Installation

```
composer require spryker/zed-ui
npm run bo:install
npm run bo:build
```
